# Ответы на вопросы для самопроверки (Лабораторная работа №3)

1. **Как зарегистрировать сервис в ASP.Net Core?**
   В файле `Program.cs` (или методе расширения) через коллекцию `builder.Services`, используя методы `AddTransient`, `AddScoped` или `AddSingleton`. Например: `builder.Services.AddScoped<IMyService, MyService>();`.

2. **Чем отличается Transient сервис от Scoped сервиса?**
   - **Transient**: Новый экземпляр сервиса создается каждый раз, когда он запрашивается из контейнера DI.
   - **Scoped**: Экземпляр сервиса создается один раз для каждого HTTP-запроса и переиспользуется в рамках этого запроса.

3. **Как внедрить сервис в метод контроллера?**
   Добавить интерфейс сервиса в качестве параметра метода действия (Action Injection) и пометить его атрибутом `[FromServices]`. Либо (чаще всего) через конструктор контроллера.

4. **Как передать данные от клиента в метод контроллера?**
   Данные могут быть переданы через:
   - Строку запроса (Query String).
   - Данные маршрута (Route Data).
   - Тело запроса (Form Data / JSON).
   - Заголовки HTTP (Headers).

5. **Где механизм привязки (Model binding) ищет нужные значения?**
   По умолчанию ищет в (в порядке приоритета): Form fields, Route values, Query string.

6. **Как получить данные из файла appsettings.json?**
   Через внедренный сервис `IConfiguration`. Например: `_configuration["Section:Key"]` или `_configuration.GetSection("Section").Get<Type>()`.

7. **Как передать данные с помощью IOptions?**
   1. Создать класс настроек (POCO).
   2. Зарегистрировать его в DI: `services.Configure<MyOptions>(Configuration.GetSection("MySection"))`.
   3. Внедрить `IOptions<MyOptions>` в конструктор потребителя.

8. **Как в коде прочитать значение строки запроса?**
   Через свойство `Request.Query`. Например: `HttpContext.Request.Query["paramName"]`.

9. **Как в теге `<a>` передать данные через tag-helper?**
   Используя атрибут `asp-route-{parameterName}`. Например: `<a asp-action="Index" asp-route-id="5">Link</a>` создаст ссылку `.../Index?id=5` (или `.../Index/5` в зависимости от маршрута).

10. **Что такое «Явное выражение Razor»?**
    Это выражение кода C#, которое начинается с `@(` и заканчивается `)`. Используется, когда нужно явно указать границы выражения, например, для вычислений или конкатенации: `Item @(i + 1)`.
