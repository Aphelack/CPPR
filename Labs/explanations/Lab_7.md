# Лабораторная работа №7: Tag Helpers и AJAX

## Новые технологии и концепции

### 1. Tag Helpers (Вспомогательные классы тегов)
- **Tag Helpers**: Компоненты в ASP.NET Core, которые позволяют серверному коду участвовать в создании и рендеринге HTML-элементов в Razor-файлах. Они выглядят как обычные HTML-атрибуты (например, `asp-controller`, `asp-action`), но обрабатываются на сервере.
- **Создание собственных Tag Helpers**: Класс должен наследоваться от `TagHelper` и переопределять метод `Process` или `ProcessAsync`.
- **TagBuilder**: Класс-помощник для программного создания HTML-тегов в коде C# (используется внутри Tag Helpers).
- **Регистрация**: Чтобы использовать Tag Helpers, их нужно подключить в `_ViewImports.cshtml` с помощью директивы `@addTagHelper`.

### 2. AJAX (Asynchronous JavaScript and XML)
- Технология обращения к серверу без перезагрузки всей страницы.
- В лабораторной используется для частичного обновления списка товаров и пейджера при переключении страниц.
- **jQuery `.load()`**: Метод JavaScript библиотеки jQuery, который загружает данные с сервера и помещает возвращенный HTML в выбранный элемент DOM.

### 3. Частичные представления (Partial Views)
- Представления (`.cshtml`), которые рендерятся внутри других представлений. Используются для повторного использования разметки и для AJAX-обновлений частей страницы.
- Возвращаются из контроллера методом `PartialView()`.

### 4. Методы расширения (Extension Methods)
- Позволяют добавлять новые методы в существующие типы без изменения их исходного кода.
- В лабораторной создается метод `IsAjaxRequest()` для класса `HttpRequest`, чтобы проверять наличие заголовка `X-Requested-With`.
