# Лабораторная работа №2: Язык Razor. Частичные представления и компоненты

## Новые технологии и концепции

### 1. Tag Helpers (Тэг-хелперы)
- Специальные атрибуты (начинаются с `asp-`), которые обрабатываются на сервере и генерируют стандартный HTML.
- **asp-controller / asp-action**: Генерируют URL для ссылки (`href`) или формы (`action`) на основе маршрутизации MVC.
- **asp-area**: Указывает область (Area) для маршрутизации.
- **asp-page**: Используется для ссылок на Razor Pages.
- **asp-items**: Заполняет элементы списка (`<select>`) данными из `IEnumerable<SelectListItem>`.

### 2. Передача данных в представление
- **ViewData**: Словарь `Dictionary<string, object>` для передачи небольших объемов данных из контроллера в представление. Требует приведения типов при чтении.
- **ViewBag**: Динамическая обертка над `ViewData`. Позволяет обращаться к данным как к свойствам объекта (`ViewBag.Title`). Не требует приведения типов, но не имеет проверки типов при компиляции.
- **Model (Модель представления)**: Строго типизированный объект, передаваемый в представление.
    - Директива `@model Type` указывает тип модели.
    - Свойство `@Model` предоставляет доступ к объекту модели.

### 3. Частичные представления (Partial Views)
- Файлы разметки Razor (обычно начинаются с `_`), которые можно переиспользовать внутри других представлений.
- Используются для выделения общих частей интерфейса (меню, футер, карточка товара).
- Подключаются с помощью тега `<partial name="_Name" />` или метода `@await Html.PartialAsync()`.

### 4. Компоненты представлений (View Components)
- Более мощный аналог частичных представлений. Состоят из двух частей:
    1. **Класс C#**: Наследуется от `ViewComponent`, содержит логику (метод `Invoke` или `InvokeAsync`).
    2. **Представление Razor**: Обычно называется `Default.cshtml`.
- Идеальны для блоков, требующих сложной логики получения данных (например, корзина покупок, облако тегов), которая не зависит от основного контроллера страницы.
- Вызываются через `@await Component.InvokeAsync("Name")`.

### 5. SelectList
- Класс, представляющий список элементов для выпадающего списка (`<select>`). Позволяет указать источник данных, поле для значения (`Value`) и поле для отображения (`Text`).
